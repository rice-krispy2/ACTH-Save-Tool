# ACNH Save Recovery Tool

This tool aims to fetch the save file from the users Appdata folder.

This can help with many things including but not limited to:
1) Migrating to a new emulator
2) Troubleshooting the "Save Data is Damaged" Error
3) Making copies of your save file

This tool does *not* work with portable installs. It was not designed for that.

Like with my ACNH Installer, I am ***not*** affiliated with Nintendo in any way.

This program contains no copyrighted code as I am merely moving files in the user's computer

To Use:
1) Download from the releases tab
2) Extract the zip
3) Run the exe
